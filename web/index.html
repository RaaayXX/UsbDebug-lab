SmartUSB Hub

快捷指令

对被测通道执行电源、数据通断及采样操作。

Hub 可独立控制 VBUS 与 D+/D-。「断电」只断 5V；「断供电·保数据」先接通数据线再断 VBUS。若 COM 口仍消失，通常是设备在掉 VBUS 后主动断开 USB（需电池 + 固件支持），而非 Hub 把数据也断了。
